## ğŸš€ Deployment

### Option 1: Separate Domain
Deploy to `www.pointwave.ng`:
```bash
npm run build
scp -r build/* user@server:/path/to/www.pointwave.ng/
```

### Option 2: Subdirectory
Deploy to `app.pointwave.ng/landing`:
```bash
# Update package.json
"homepage": "/landing"

# Build and deploy
npm run build
scp -r build/* user@server:/path/to/app.pointwave.ng/public/landing/
```
